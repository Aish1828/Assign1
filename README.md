# MAISON - Luxury Fashion E-Commerce Website

## Overview

MAISON is a modern luxury fashion e-commerce website built using React. The project focuses on creating an elegant and visually appealing user interface inspired by premium fashion brands. It showcases collections, products, and brand information through a clean and sophisticated design.

## Features

* Modern luxury fashion website design
* Responsive navigation bar
* Interactive homepage hero section
* Collections showcase page
* Product shop page with category filters
* About page with brand story
* Professional typography and layout
* Responsive user interface
* Multi-page navigation using React Router

## Pages

### Home

Features a premium hero section showcasing the latest fashion collection with elegant branding and call-to-action buttons.

### Collections

Displays curated fashion collections with detailed descriptions and seasonal highlights.

### Shop

Showcases available products with category-based organization and product gallery layouts.

### About

Presents the MAISON brand philosophy, history, and commitment to quality fashion.

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd maison
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5175
```
